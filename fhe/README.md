# FHE Modülü Yol Haritası

Tam homomorfik şifreleme (FHE) entegrasyonu bu klasörde geliştirilecektir. İlk aşamada mesaj içerikleri açık metin olarak saklanırken, sonraki sprintlerde Zama'nın TFHE kitaplıkları ile şifreli mesaj desteği hazırlanacaktır.

> Not: Bu klasör henüz derlenebilir Rust kodu içermiyor. `cargo init` ile yeni bir crate oluşturup Zama FHE SDK'sını eklemeye başlayabilirsiniz.
